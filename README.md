# Gemini Image Bot (Nana Banana)

Telegram бот для генерации и редактирования изображений через Google Gemini 2.5 Flash Image.

## Возможности
- 🎨 Генерация изображений по тексту
- 📝 Редактирование загруженных фото
- 🖼️ Композиция из нескольких изображений
- 📊 Личная статистика и история

## Установка
1. Клонируйте репозиторий
2. Создайте виртуальное окружение
3. Установите зависимости: `pip install -r requirements.txt`
4. Скопируйте `.env.example` в `.env` и заполните ключи
5. Запустите: `python main.py`

## Команды
- `/start` - Главное меню
- `/help` - Справка
- `/stats` - Моя статистика
- `/history` - История генераций
